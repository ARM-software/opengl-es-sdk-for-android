bool useFence = true
static

A sample which illustrates the use of sync objects to synchronise the use of shared objects between multiple contexts in multiple threads.

A cube is rendered in the main application thread using a texture which is updated in a second thread. Each thread has its own rendering context and both contexts share their EGL objects. Sync objects are used to synchronise the access to shared data.
